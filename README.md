# Salamander Space Marine Chatbot

A desktop chatbot that embodies the noble and fierce spirit of a Salamander Space Marine from Warhammer 40k.

## Features
- Floating window interface
- Chat history storage
- Powered by Ollama's deepseek-r1 model
- Authentic Salamander Space Marine persona

## Prerequisites
1. Install Ollama from https://ollama.ai/
2. Pull the deepseek-r1 model:
   ```
   ollama pull deepseek-r1
   ```

## Setup
1. Install Python 3.8 or higher
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Running the Application
1. Make sure Ollama is running in the background
2. Run the application:
   ```
   python main.py
   ```

## Usage
- Type your message in the input box and press Enter or click Send
- The chatbot will respond in the style of a Salamander Space Marine
- Chat history is automatically saved locally
- The window can be dragged around the screen
